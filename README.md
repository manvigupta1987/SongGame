# SongGame
This app is a game where user can guess the composer by listening the song. This app uses ExoPlayer to play the songs.

## Libraries
* [ExoPlayer](https://github.com/google/ExoPlayer)

## UI Component
* [Button](https://developer.android.com/reference/android/widget/Button.html)
* [ConstraintLayout](https://developer.android.com/training/constraint-layout/index.html)
* [LinearLayoutManager](https://developer.android.com/reference/android/support/v7/widget/LinearLayoutManager.html)
* [SeekBar](https://developer.android.com/reference/android/widget/SeekBar.html)
